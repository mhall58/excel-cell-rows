Excel: Split Cell to Rows

This utility reads the content of one column and splits the value by the selected delimiter. A new row with a copy of all other columns is created for each value.

For example...

This:
/ A B C
1 Value 1 Value 2 Value 3,Value 4

To This:
/ A B C
1 Value 1 Value 2 Value 3
2 Value 1 Value 2 Value 4
This is how we know when to stop. Use a column that is always populated.
Note: Data is processed in memory only. File is not saved on the server. Using this service is not logged.
